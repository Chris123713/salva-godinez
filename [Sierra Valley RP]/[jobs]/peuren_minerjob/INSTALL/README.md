2. Add items to your inventory config/database.

3. Check if the peuren_lib/shared/config.lua is setup correctly to your server and your dependencies.

4. Start the system in your server.cfg, this can be achieved by adding this line in your server.cfg: ensure peuren_minerjob

5. Restart your server and enjoy our system!
