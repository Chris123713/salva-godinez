# Elevator Sound Files

This directory should contain the following sound files in `.ogg` format:

## Required Sound Files:

1. **elevator_ding.ogg**
   - Elevator arrival "ding" sound
   - Recommended: Short bell/chime sound (0.5-1 second)
   - Plays when elevator arrives at a floor

2. **elevator_doorOpen.ogg**
   - Door opening sound
   - Recommended: Mechanical door opening sound (1-2 seconds)
   - Plays when elevator doors open

3. **elevator_doorClose.ogg**
   - Door closing sound
   - Recommended: Mechanical door closing sound (1-2 seconds)
   - Plays when elevator doors close

4. **elevator_movement.ogg**
   - Elevator movement/motor sound
   - Recommended: Continuous mechanical hum/motor sound (3-5 seconds, will loop)
   - Plays during elevator travel

## Where to Find Free Sounds:

- **Freesound.org** - Free sound effects library
- **Zapsplat.com** - Free sound effects (registration required)
- **Soundbible.com** - Royalty-free sounds
- **YouTube Audio Library** - Download and convert to OGG

## Converting to OGG:

If you have MP3 or WAV files, you can convert them to OGG using:
- **Online:** https://convertio.co/mp3-ogg/
- **Software:** Audacity (free), FFmpeg

## File Size Recommendations:

- Keep each file under 100KB for best performance
- Use mono audio (not stereo) to reduce file size
- Sample rate: 44.1kHz or 48kHz
- Bitrate: 64-128 kbps

## Installation:

1. Download or create your sound files
2. Convert to .ogg format if needed
3. Rename to match the exact filenames listed above
4. Place all files in this directory
5. Restart the resource

## Note:

The elevator system will work without sounds, but sound effects enhance immersion significantly. If sound files are missing, the script will continue to function normally without audio.
